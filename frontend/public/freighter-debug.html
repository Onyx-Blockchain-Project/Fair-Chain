🛠️ Manual Fixes

Extension Fixes

  1. Click the puzzle icon 🧩 in toolbar
  2. Find "Freighter" in the list
  3. Make sure the toggle is ON (blue)
  4. Click the pin icon 📌 to show it in toolbar
  5. Refresh this page (F5)

Browser Fixes

  1. Try a different browser (Chrome/Firefox)
  2. Disable other wallet extensions temporarily
  3. Clear browser cache
  4. Try incognito mode
  5. Restart browser completely
